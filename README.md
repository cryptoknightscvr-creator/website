# CryptoKnights Official Website

The official website for CryptoKnights, the Computer Science and Engineering (Cyber Security) club at CVR College of Engineering.

## Tech Stack
- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- Framer Motion
- GSAP & Lenis Smooth Scroll
- Lucide React Icons

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy the environment variables:
   ```bash
   cp .env.example .env.local
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
